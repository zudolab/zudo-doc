Willkommen bei **zudo-doc** — einem minimalen und schnellen Dokumentations-Framework, gebaut mit Astro und MDX.

## Warum zudo-doc?

- **Schneller Entwicklungsserver** — Vite-basiert, Seiten werden bei Bedarf kompiliert
- **MDX-Unterstützung** — Dokumente mit JSX-Komponenten schreiben
- **Tailwind CSS** — Utility-First-Styling
- **Statischer Export** — Generiert reines HTML, standardmäßig kein JS
- **Minimale Basis** — Erweiterbar mit eigenen Komponenten

## Schnellstart

```bash
pnpm install
pnpm dev
```

Öffnen Sie [http://localhost:4321](http://localhost:4321), um die Dokumentation anzuzeigen.
