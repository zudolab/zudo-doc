Welcome to **zudo-doc v1.0** documentation.

This is the first stable release of zudo-doc, featuring:

- Astro-based static documentation framework
- MDX support with admonitions
- 16-color terminal palette system
- Client-side search with MiniSearch
- Multi-locale support (i18n)
