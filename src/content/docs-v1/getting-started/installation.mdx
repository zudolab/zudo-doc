## Prerequisites

- Node.js 18 or later
- pnpm 8 or later

## Quick Start

```bash
git clone https://github.com/zudolab/zudo-doc.git
cd zudo-doc
pnpm install
pnpm dev
```

Open `http://localhost:4321` to see your documentation site.
