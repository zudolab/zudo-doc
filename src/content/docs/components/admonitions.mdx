Admonitions are callout blocks for highlighting important information. All five types are globally available — no imports needed.

## Note

<Note>
This is a note — use it for general information and tips.
</Note>

<Note title="Custom Title">
Notes are great for calling out important information readers shouldn't miss.
</Note>

```mdx
<Note>
This is a note — use it for general information and tips.
</Note>

<Note title="Custom Title">
Notes are great for calling out important information readers shouldn't miss.
</Note>
```

## Directive Syntax

In addition to the JSX component syntax shown above, you can use Docusaurus-style directive syntax. Both syntaxes are globally available — no imports needed.

```mdx
:::note
Content here.
:::

:::warning[Custom Title]
Content here.
:::
```

The directive syntax supports all five types (`note`, `tip`, `info`, `warning`, `danger`) and an optional title in square brackets.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | string | Type name (e.g., "Note") | Override the default title text |

## Color Reference

Each admonition type uses a semantic color token for its border and title color:

| Type | Color Token | Typical Color |
|------|------------|---------------|
| Note | accent | Blue |
| Tip | success | Green |
| Info | info | Cyan |
| Warning | warning | Yellow |
| Danger | danger | Red |
