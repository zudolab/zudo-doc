## 概要

サイドバーフィルターは、サイドバー上部に表示されるテキスト入力欄で、入力に応じてナビゲーション項目をリアルタイムに絞り込みます。ページ数の多いドキュメントサイトで、サイドバーツリー全体をスクロールせずに目的のページを素早く見つけられます。

`create-zudo-doc` で作成されたプロジェクトでは、サイドバーフィルターはデフォルトで有効です。

## 仕組み

フィルター入力欄はサイドバーのナビゲーションツリーの上に表示されます。入力すると:

- カテゴリ名と個々のページタイトルの両方が検索対象になります
- 大文字・小文字を区別しません
- カテゴリ名がマッチした場合、そのカテゴリの子ページがすべて表示されます
- 子ページのみがマッチした場合、親カテゴリとマッチした子ページだけが表示されます
- フィルタリング中はマッチしたカテゴリがすべて自動的に展開されます
- 入力をクリアすると、サイドバーツリー全体が復元されます

## キーボードショートカット

ページ上のどこからでも `Ctrl+/`（Windows/Linux）または `Cmd+/`（macOS）でフィルター入力欄にフォーカスできます。サイドバーが表示されている場合（デスクトップビューポートまたはモバイルサイドバーを開いた状態）に動作します。

## 有効化と無効化

### create-zudo-doc での設定

サイドバーフィルターはデフォルトで有効です。プロジェクト作成時に無効にするには:

```bash
pnpm create zudo-doc my-docs --no-sidebar-filter
```

明示的に有効にするには:

```bash
pnpm create zudo-doc my-docs --sidebar-filter
```

### 既存プロジェクトでの設定

サイドバーフィルターは `SidebarTree` React コンポーネント（`src/components/sidebar-tree.tsx`）に組み込まれています。サイドバーの一部として常にレンダリングされ、`settings.ts` に個別のトグル設定はありません。スキャフォールドされたプロジェクトからフィルターを削除するには、コンポーネントを直接編集する必要があります。

<Tip>
ほとんどのドキュメントサイトでは、ページ数が少なくてもサイドバーフィルターは便利です。コンテンツが数十ページを超えると不可欠になります。
</Tip>
