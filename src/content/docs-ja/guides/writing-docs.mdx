## ドキュメントの作成

`src/content/docs/` に `.mdx` ファイルを作成します。フロントマターでメタデータを指定してください：

```mdx
---
title: マイページ
sidebar_position: 1
---

ここにコンテンツを記述します。
```

## ディレクトリ構造

ディレクトリを使ってドキュメントをカテゴリに整理します：

```
src/content/docs/
  getting-started/
    introduction.mdx
    installation.mdx
  guides/
    writing-docs.mdx
```

各ディレクトリは自動的にサイドバーのカテゴリになります。

## MDXの機能

ドキュメント内でReactコンポーネントを使用できます。インタラクティブなコンポーネントはアイランドとしてハイドレートされ、残りは純粋なHTMLとして配信されます。
