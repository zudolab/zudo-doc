## 0.1.0

zudo-docの初回リリース。

### 機能

- Astro 6とMDXコンテンツコレクション
- 16色パレットによるTailwind CSS v4デザイントークンシステム
- ファイル構造からの自動生成ツリーによるサイドバーナビゲーション
- i18nサポート（英語・日本語）
- アドモニションコンポーネント（Note、Tip、Info、Warning、Danger）
- Shikiによるコードハイライト
- スクロールスパイ付き目次
- 複数の組み込みテーマによるカラースキーム切り替え
- 検索機能
