# S4 Header Navigation & Right Items — Audit

**Build base:** branch `zfb-feature-audit/s4-header`, built from `pnpm build` on 2026-05-04.  
**Pages inspected:** `dist/docs/getting-started/index.html`, `dist/ja/docs/getting-started/index.html`.  
**Sources read:** `packages/zudo-doc-v2/src/header/header.tsx`, `nav-active.ts`, `nav-overflow-script.ts`, `pages/lib/_header-with-defaults.tsx`, `src/config/settings.ts`, `src/utils/header-right-items.ts`, `src/components/sidebar-toggle.tsx`.

**Conclusion:** Navigation items, active-state logic, dropdown markup, overflow "···" mechanism, and mobile burger are all correct. One functional bug found: the desktop language-switcher slot is never populated — the `languageSwitcher` prop is not passed to `<Header>` from `_header-with-defaults.tsx`, so it renders as an empty `<div>`. One doc discrepancy found: `header-right-items.mdx` shows an outdated default array that omits `search` and lists items in a different order than the actual `settings.ts`.
