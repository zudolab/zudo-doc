---
title: Code blocks with titles
---

Plain code block, no title:

```js
const x = 1;
```

Code block with a title:

```ts title="src/config.ts"
export const x: number = 42;
```

Bash with title:

```bash title="install.sh"
pnpm install
pnpm build
```

JSON with title:

```json title="package.json"
{
  "name": "demo"
}
```
